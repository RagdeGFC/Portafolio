HTML

CSS3

En CSS3, manejo tanto el diseño básico como avanzado, aplicando estilos visuales y creando interfaces atractivas y fluidas. Uso de técnicas como flexbox y grid para layouts responsivos, así como animaciones y transiciones para mejorar la interactividad y usabilidad de los sitios web.

Javascript

React

Proyectos

Proyecto 1: Frases del Caballero Oscuro (React, CSS3, JavaScript, HTML)

Desarrollé una aplicación interactiva que genera frases aleatorias del universo de Batman. Implementé la funcionalidad usando React para el manejo de estados y componentes, CSS3 para un diseño oscuro y temático, y JavaScript para la lógica de generación aleatoria de frases. La aplicación incluye animaciones suaves y es completamente responsive.

Proyecto 2: Aplicación del clima (React, CSS3, JavaScript, HTML)

Creé una aplicación del clima que consume una API de pronóstico meteorológico. Utilicé React para crear componentes reutilizables, implementé geolocalización para detectar la ubicación del usuario, y apliqué CSS3 para diseños dinámicos que cambian según las condiciones climáticas. La app muestra temperatura, humedad, velocidad del viento y pronóstico por horas.

Proyecto 3: Aplicación de Rick & Morty (React, CSS3, JavaScript, HTML)

Desarrollé una aplicación web que consume la API de Rick & Morty, permitiendo explorar personajes, ubicaciones y episodios de la serie. Implementé filtros de búsqueda avanzados, paginación y diseño responsive usando React. La interfaz incluye tarjetas interactivas con información detallada de cada personaje y animaciones fluidas en CSS3.

Proyecto 4: Aplicación de Pokémon (React, CSS3, JavaScript, HTML)

Implementé una Pokédex completa utilizando la PokeAPI, donde los usuarios pueden explorar y buscar Pokémon. La aplicación incluye detalles de cada Pokémon como estadísticas, tipos, evoluciones y movimientos. Utilicé React Router para la navegación, implementé un sistema de filtrado por tipos y regiones, y diseñé una interfaz responsive con CSS3 que mantiene la estética clásica de Pokémon.
